# Smart-Eye

Project code lives in the **`Smart Eye/`** directory (Flask app, templates, tests).

- **Setup and run:** see [Smart Eye/README.md](Smart Eye/README.md)
- **Repository:** [github.com/SayyadChandan/Smart-Eye](https://github.com/SayyadChandan/Smart-Eye)

```bash
cd "Smart Eye"
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
